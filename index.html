Process

task subtasks benefit
research and content strategy interview client, content inventory, model content this is design, not art. we need something work with here.
wireframes establish componentes in a mobile layout, then wider layouts define components and UX
style tiles interview client, present colors, type, textures/imagery, key components establish a visual direction that speaks to the brand ID and project tone
visual design iteratively refine the modules, page, and layouts apply the winning style tile direction to the more structural design
style guide collect components into one place to evaluate the design system as a whole
